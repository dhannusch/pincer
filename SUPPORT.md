# Support

## Supported Environments (Initial OSS Release)

- Node.js LTS (20.x and newer LTS)
- Linux and macOS

Windows is best-effort until explicitly added to the support matrix.

## Where to Get Help

- Usage questions and bug reports: GitHub Issues
- Feature proposals: GitHub Issues
- Security concerns: `SECURITY.md`

## Troubleshooting Basics

```bash
npm run typecheck
npm test
```

For Cloudflare issues:

```bash
npx wrangler whoami
npx wrangler --version
```
